1990 - World Wide Web

Tim Berners-Lee crea la Web en el CERN.

Información Detallada:

Tim Berners-Lee, un físico británico, inventó la World Wide Web mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear) en Suiza. Creó el primer navegador web llamado WorldWideWeb y el primer servidor web. El 6 de agosto de 1991, el sitio web del CERN se convirtió en el primer sitio web del mundo.

Características Principales:

  • Invención del HTML (HyperText Markup Language)
  • Creación de HTTP (HyperText Transfer Protocol)
  • Primer navegador web: WorldWideWeb
  • Sistema de hipertextos accesible públicamente
